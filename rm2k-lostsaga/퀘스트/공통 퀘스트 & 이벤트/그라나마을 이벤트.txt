<그라나마을 이벤트>

마나코어(난이도:★★★☆☆, 적정레벨 전사계열 28, 법사계열 32, 사제계열 34 이상)
그라나마을의 북서쪽에 가면 쿤이 정화된핵을 가지고있냐고 묻습니다.
정화된핵은 다크나이트의핵을 로저스국의 누군가가 정화시켜준다고 하는데,
그럼 일단 다크나이트의핵을 얻어야겠죠?

다크나이트의핵은 신규 던전인 그라나마을 남서쪽에 있는 지하던전의 진보스에게서 얻을 수 있습니다.
진보스 다크나이트와 싸우는 방법은 보스 듀라한을 잡은 뒤, 다시 보스방으로 들어와 계속 마법진에 가면 됩니다.

다크나이트를 잡아서 핵을 얻으셨다면, 그라나마을의 북서쪽 쿤에게 갑시다.
쿤에게 가서 대화를 하면, 로저스국에서 핵을 정화시켜줄 사람이 생깁니다.

로저스국 수비대장 "데니온"또는 "모로나"의 오른쪽으로 길따라 쭉 가다보면
핵을 정화시켜줄 프레야가 있습니다.
프레야는 다크나이트의핵, 니케의수호, 기억의돌을 각각 1개씩 가져다주면 정화시켜준답니다.

정화된핵을 가지고 다시 그라나마을 쿤에게 돌아갑시다.
그럼 암흑저서를 가지고있냐고 묻습니다.
암흑저서와 정화된핵을 주면, 장신구 '코어'를 줍니다.

만약 지하던전 보스 듀라한을 잡고 나온 '마나의핵'이 있다면,
다시 로저스국 수비대장 오른쪽으로 길따라 프레야가 있던 곳으로 갑시다.
그럼 프레야 대신 쿤이 다시 있는데, 마나의핵과 코어를 합쳐서 '마나코어'라는 아이템을 만들어줍니다.

다크나이트의핵 : 지하던전 - 다크나이트
니케의수호 : 로저스국 중앙광장 - 피리(퀘스트), 본토던전4층 - 다크 이플립트리더,
마왕의침소 - 뱀파이어, 언뱀파이어, 천상계1층 - 배회하는 다크엘프,
천상계3층 - 쉐도우베얼, 천상계4층 - 미노타사우르스
(어디서 나오는지 제보 바람) - 메리어울프
기억의돌 : 로저스국 중앙광장 - 피리(판매) 석화골렘
암흑저서 : 태초의산
마나의핵 : 지하던전 - 듀라한

정화된핵 : 로저스국 - 프레야(다크나이트의 핵, 니케의수호, 기억의돌로 정화)
코어 : 그라나마을 - 쿤(정화된핵, 암흑저서로 조합)
마나코어 : 로저스국 - 쿤(코어, 마나의핵으로 조합)

필요물품 : 다크나이트의핵 1개 + 니케의수호1개 + 기억의돌 1개 ---> 정화된핵 1개 + 암흑저서 1개
 ---> 코어 1개 + 마나의핵1개 ---> 마나코어 1개
보상 : 코어 / 마나코어
